# Personal Request Desk

## Motivation

Organizations often make people fill forms, open tickets, verify identity, and wait. But when those same organizations want something from a person, they may call directly, interrupt immediately, and provide little context.

Personal Request Desk is a small tool for balancing that unfairness.

If a company, government office, school, platform, landlord, or organization requires process from you, you can require process from them too.

The boundary is simple:

> Please submit a ticket first.

This is not about being rude. It is about making the relationship more equal.

## 动机

公司、政府部门、学校、平台、房东或其他组织，常常要求个人填表、提交工单、验证身份，然后等待处理。

但当这些组织反过来向个人提出要求时，却可能直接打电话、立即打断、不给清晰背景，也不给工单编号。

Personal Request Desk 是一个小工具，用来抵消这种不公平。

如果组织要求你遵守流程，你也可以要求他们遵守流程。

边界很简单：

> 请先提交工单。

这不是为了无礼，而是为了让关系更平等。

## Run

```bash
python3 server.py --host 127.0.0.1 --port 8000
```

Open `http://127.0.0.1:8000/`.
